# Keymap Visualizations

This folder contains auto-generated SVG visualizations of the Sofle keyboard keymap, created by [keymap-drawer](https://github.com/caksoylar/keymap-drawer).

## Viewing

SVG files can be opened directly in any modern web browser. They are vector graphics, so they scale to any size without losing quality.

## Generating a PDF

1. Open the SVG file in your browser (e.g. Chrome, Firefox, Edge)
2. Press `Ctrl+P` (or `Cmd+P` on Mac) to open the print dialog
3. Choose **"Save as PDF"** as the destination
4. Click **Save** — done! 🎉

## Auto-generation

The SVGs are automatically regenerated by the [Draw ZMK keymaps](./../.github/workflows/draw-keymaps.yml) GitHub Actions workflow whenever changes are pushed to:
- `config/*.keymap`
- `config/*.dtsi`
- `keymap_drawer.config.yaml`

You can also trigger the workflow manually via **Actions → Draw ZMK keymaps → Run workflow**.
